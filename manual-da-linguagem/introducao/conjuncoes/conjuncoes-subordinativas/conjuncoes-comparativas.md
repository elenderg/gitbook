# Conjunções comparativas

As conjunções subordinativas adverbiais **comparativas** estabelecem uma correspondência, um paralelo em relação a oração independente.&#x20;

**Exemplos:**&#x20;

* como
* assim como
* como se
* bem como
* que nem
* mais **que**
* menos **que**
* maior **que**
* menor **que**
* melhor **que**
* pior **que**
* mais **do que**
* menos **do que**
* maior **do que**
* menor **do que**
* melhor **do que**
* pior **do que**
* tal qual
